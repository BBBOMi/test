2> 기본 자료형과 참조형

1. Stack Segment
	- 함수(메소드) 호출과 관련된 메모리
	- 지역변수, 파라미터 변수
	
2. Heap Segment
	- new, malloc
	- 동적으로 생성된 메모리
	- Java의 참조형 객체(class, 배열, enumeration)
	
3. Data Segment
	- 앱이 시작될 때 생성되어서 종료될 때까지 유지되는 변수
	- c의 전역변수, java의 static 멤버 변수, static 지역변수

4. 객체 구조
-> 객체 비교 무조건 equls비교

-> == 비교는 idetity비교 이므로 사용할 때 주의

-> equls 메소드를 재정의하지 않으면 Object클래스의 equls메소드를 상속받아 사용하는 것이라
	사용할 때엔 반드시 재정의하여 사용하여야 한다 (Object의 equls메소드는 identity 비교이기 때문)
